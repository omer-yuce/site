== Description ==
Browse from 210+ animated SVG icons for Elementor Page Builder for your next project.

Visit [our website](http://animated-svg.com/) to view icon library and features.

https://youtu.be/XGfk7szFPyQ

Features:

- 3 types of animations
- Customizable stroke and background colors
- Adjustable scaling of icons to any size
- Custom setting of animation duration time
- Option to upload and animate your own SVG icon
- Variety of in and out animation effects
- On hover interaction option
- Ability to edit icons in any text editor

== Installation ==
From within WordPress' dashboard:

1. Go to Plugins -> Add New
2. Search for "Animated SVG"
3. Click "Install"
4. Click "Activate"
5. Open Elementor > Start using widgets

Manually via FTP:

1. Upload the folder 'animated-svg/' into the '/wp-content/plugins/' directory
2. Activate the plugin through the Plugins tab in WordPress
3. Navigate to Elementor > Start using widgets

== Frequently Asked Questions ==

I can't upload my own images, what should I do?
- In rare cases hosting providers restrict the upload of SVG files. You should install an SVG support plugin like Easy SVG support https://wordpress.org/plugins/easy-svg/ to solve the issue.

Do I need Elementor Pro to use the widgets?
- No, the plugin works fine with the free version of Elementor.

Can I use the icons without limitations?
- Yes, you may display the icons on your website. The source code of each SVG file contains the list of credits and/or the license copyright information. See license.txt for more information.
